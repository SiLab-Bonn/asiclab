Checksum notes

https://getfedora.org/en/security/

RPM files are signed by 'GPG signatures'; be sure to 'verify signatures'

Fedora provides 'current keys', for example: Fedora 32 id: 4096R/12C944D0 2019-08-12 Fingerprint: 97A1 AE57 C3A2 372C CA3A 4ABA 6C13 026D 12C9 44D0

There a fedora GPG keys, which you import to GPG: $ curl https://getfedora.org/static/fedora.gpg | gpg --import This does: gpg: key 6C13026D12C944D0: public key "Fedora (32) fedora-32-primary@fedoraproject.org" imported

Then there are also checksum files, which you can get on the page above, or here: $ curl https://getfedora.org/static/fedora.gpg | gpg --import
